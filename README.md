## Descrição

&emsp; Desenvolver uma aplicação web simples de uma bilheteria de cinema com as seguintes funcionalidades:

- Cadastro de filmes;
- Cadastro de salas;
- Cadastro de sessões e horários e
- Venda de ingressos

O projeto foi desenvolvido por meio de:

<p> Front-end:
    <ul>
        <li>HTML</li>
        <li>CSS</li>
        <li>JavaScript</li>
    </ul>

<p> Back-end:
    <ul>
        <li>Node.js</li>
        <li>C#</li>
    </ul>

<p> Banco de Dados:
    <ul>
        <li>MySQL</li>
        <li>SQLite</li>
    </ul>
